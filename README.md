# Sync 727 - מערכת ניהול קבוצה

מערכת לניהול קבוצת תעופה וירטואלית, הכוללת מעקב אחרי טיסות, ניהול משתמשים, וסטטיסטיקות.

## תכונות עיקריות
- **ניהול משתמשים:** הרשמה, התחברות, ופרופיל אישי.
- **מעקב טיסות:** רישום טיסות, צפייה בהיסטוריה, וסטטיסטיקות אישיות.
- **לוח בקרה (Dashboard):** תצוגה גרפית של נתונים וסטטיסטיקות כלליות.
- **התראות:** מערכת התראות פנימית למשתמשים.
- **PWA:** תמיכה בהתקנה כאפליקציה במכשירים ניידים.

## התקנה והרצה מקומית

1.  שכפל את המאגר:
    ```bash
    git clone https://github.com/YOUR_USERNAME/sync-727.git
    cd sync-727
    ```

2.  התקן תלויות:
    ```bash
    npm install
    ```

3.  הגדר משתני סביבה:
    - העתק את הקובץ `.env.example` ל-`.env`
    - מלא את הערכים החסרים (Supabase, Firebase, Gemini API וכו')

4.  הרץ את השרת:
    ```bash
    npm run dev
    ```

## פריסה (Deployment)

הפרויקט מוכן לפריסה ב-Netlify. ראה קובץ `DEPLOY.md` להוראות מלאות.
